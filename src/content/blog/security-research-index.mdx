This is my working index for security research and exploit-development material.
It is intentionally practical: use the search box to jump from a researcher to
a technique, mitigation, tool, or low-level API reference without digging
through bookmarks.

The first pass is anchored on [afflicted.sh](https://afflicted.sh/blog/) and
three posts worth keeping close:

- [The KASLR graveyard](https://afflicted.sh/blog/posts/kaslr-graveyard.html)
- [amwrtdrv: AOMEI Backupper hands every local user the disk](https://afflicted.sh/blog/posts/amwrtdrv-aomei-lpe.html)
- [SignalRGB: standard user to SYSTEM](https://afflicted.sh/blog/posts/signalrgb-service-lpe.html)

The researcher table separates people and blogs from individual resources. The
second table collects techniques, mitigation and evasion material, exploit case
studies, and tooling. Stable API and structure documentation lives in a third
technical-reference table so it does not get mixed into the reading queue. This
index will grow as I work through more research.

## Scope and attribution

Descriptions are short navigation notes, not substitutes for the original
research. Follow the source links for technical details and credit. Inclusion
does not imply that every linked researcher authored every technique listed;
the source column identifies the relevant publication.
